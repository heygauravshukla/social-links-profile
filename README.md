### What I Learned

#### In HTML

- **`<figure>` and `<figcaption>` tags**:

  ```html
  <figure>
    <img ... />
    <figcaption>...</figcaption>
  </figure>
  ```

  These tags can be used to display an image along with its caption more semantically.

- **`<address>` tag**:

  ```html
  <address>...</address>
  ```

  This tag can be used to display address or location-related information.

#### In CSS

- **`@font-face`**:

  ```css
  @font-face {
    font-display: ...;
    font-family: ...;
    font-style: ...;
    font-weight: ...;
    src: url("...");
  }
  ```
