### What I Learned

#### In HTML

- **`<figure>` and `<figcaption>` tags**:

  ```html
  <figure>
    <img ... />
    <figcaption>...</figcaption>
  </figure>
  ```

  These tags can display an image along with its caption in a more semantic way.

- **`<address>` tag**:

  ```html
  <address>...</address>
  ```

  This tag can be used to display address or location-related information.

#### In CSS

- **`@font-face`**:

  ```css
  @font-face {
    font-display: ...;
    font-family: ...;
    font-style: ...;
    font-weight: ...;
    src: url("...");
  }
  ```
